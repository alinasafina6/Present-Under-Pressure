# Present-Under-Pressure
Data hackathon project in R to investigate the relationship between class attendance and stress level.

Question of Interest: Does a statistically significant relationship exist between class attendance and perceived stress levels?

While some students view skipping a lecture as a necessary "mental health break," others find that missing class creates even more stress due to the material that is missed. So, which of these opinions is the best approach? With this analysis, I wanted to investigate the relationship between class attendance and stress level to finally answer this question of interest.
